The easy way to more functionality

rutema_elements is a collection of modules containing element implementations available as a gem.
It is the easiest way to add functionality to a rutema parser.
Here's how to do it:

Choose the module(s) that provide the elements you need. For example the Rutema::Elements::Standard module defines the elements wait and fail.
Now derive your parser from Rutema::MinimalXMLParser and include the module: 

require 'rubygems'
require 'rutema/system'
require 'rutema/elements/general'
class ExampleParser < MinimalXMLParser
  include Rutema::Elements::Standard
end

And that's it! The implementations provided aim to be very flexible, with reasonable defaults and as many possibilities of parametrization as possible. Extended documentation, including examples is provided in the gem's rdocs
