Installing rutema
Version>=1.0.0

Installation

rutema's installation is simpler when using the gems. Still there are some dependencies that are not automatically included as they present some difficulties (mostly under windows).

rutema

Start by installing rutema using the gem facilities:

sudo gem i rutema

If you are going to use the ActiveRecordReporter (and you will want to use it - much of rutema's value comes from archiving test results) you will need sqlite3:

sudo gem i sqlite3-ruby --version="1.2.2"

Windows users should make sure to have the sqlite3 dll in windows/system32. Installing the latest version of sqlite3-ruby on windows is not straightforward as there is no precompiled win32 gem. At the time of writing the latest precompiled sqlite3-ruby version is 1.2.2

This takes care of the base rutema system, allowing you to create and run tests, archive the results and produce email and text reports.

rutemax -v will tell you the version of rutema you have installed

rutemaweb

The rutemaweb gem offers a simple web GUI implemented with Ramaze for test result visualisation:

sudo gem i rutemaweb

On windows, make sure that the win32-console gem is installed (gem i win32-console) otherwise Ramaze will exit with an error.

rutemaweb -v will tell you the version of rutemaweb you have installed

rutema elements

rutema elements is a collection of element parsing implementations that simplifies parser development based on Rutema::BaseXMLParser
sudo gem i rutema_elements will install it in your system.

Some implementations target Windows tools (like sqlcmd or mstest) but the code is in pure Ruby with no platform dependencies.
