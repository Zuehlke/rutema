Nesting rutema scenarios

The include_scenario element

rutema added the include_scenario element in the base parser of version 0.9.0
Using it, you can nest scenarios and reuse reoccuring step sequences in all parsers derived from BaseXMLParser (this includes ExtensibleXMLParser and MinimalXMLParser).

<include_scenario file="path/to/include.scenario" />

The file to be included should contain just the scenario element (with it's subelements):

<scenario>...</scenario>

T005.spec shows how a sampleinclude.scenario is included.
