Upgrading to rutema v1.0.0
Between the 0.9.3 and the 1.0.0 releases of rutema there has been a change in the database schema.
This change adds more information to the scenario entries, capturing the titles and descriptions associated with them.
The rutema 1.0.0 gem includes rutema_upgrader, a small utility that will perform the migration. Simply call rutema_upgrader config_file.rutema to perform the migration.

The migration adds the columns 'title' and 'description' to the scenarios table. The script uses the configuration to parse all specifications and add the appropriate title and description values. It looks for the ActiveRecordReporter configuration to determine which database to migrate.
