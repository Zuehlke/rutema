rutema 0.9.4 is out!

rutemaweb 0.9.4 has been released!

rutemaweb now offers you useful information in pretty colors.
The aesthetics might be debatable, but the usefulness of the information is not!

The new statistics section in this version offers test run information on a time axis.
It gives you the number of tests per run grouped as succesful and failed tests. Take the following screenshot for example

If you ever see this trend in your project be worried...be very worried!

To get the gem just do: gem install rutemaweb
rutemaweb now depends on gruff and it will pull the gruff gem as a dependency, but gruff does not define it's dependencies so you will probably have to install RMagick as well.
Don't worry, rutemaweb will function even if gruff/RMagick are not properly installed - you just won't get the pretty colors.

You can also get the gem directly from Rubyforge
