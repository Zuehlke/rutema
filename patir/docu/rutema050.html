rutema 0.5.0 is out!

rutema 0.5.0 has been released!
More documentation, more examples, a new web frontend.

0.5.0 is not Camping anymore.
Instead the web fronted (rutemaweb) is implemented using Ramaze
This also solves some problems with the webrick code in Camping 1.5

A handful of bugs have been also fixed, most of them in the silent 0.4.3 release:

  • rutemah now actually prints the report
  • Nested directory changes now function correctly (this also raises the patir dependency to version 0.5.1 )
  • fixed a couple of bugs relating to using relative paths in the configruation
  • Distro Test now includes an example using the active record reporter

The biggest news is that rutema can now test itself! There is a test specification with rutemax and rutemah calls based on the distro test used as a general smoke/system test.

To get the gem just do: gem install rutema

You can also get the gem directly from Rubyforge
Get started with rutema by reading about the Distro Test example
