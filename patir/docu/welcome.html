Welcome to patir

We are very proud to welcome you to patir, a project that aims to provide you with a set of tools that will make your software developing life just a little bit easier.
The way we will organise this project is to use patir as an umbrella project and create a subproject for each of the tools we develop.
All code found to be used throughout the tools will be maintained in the patir library in good old DRY fashion.

patir (pronounced pat-IR) is not just an acronym (Project Automation Tools in Ruby): It means "father" in greek ("πατήρ").
So welcome to the father of all our projects :).
