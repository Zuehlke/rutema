rutema 0.3 is out!

rutema 0.3 has been released!
This release adds rutemah to the tools installed plus an ActiveRecord reporter.

The ActiveRecordReporter when used in a rutemax configuration will save all test run results in a SQLite database.
You can then peek into that database with rutemah. For this first release rutemah has a rather limited set of functionality:
It can display all the test results in the database or give you the complete history for one specific test

This new release add three new dependencies to rutema: activerecord, sqlite3 and ruport.

Up and coming features

One of the goals for rutemah is to enable the re-creation of reports (using the same reporters as in your rutemax configuration)

ActiveRecordReporter will be extended so that all ActiveRecord adapters are usable (not just SQLite)

Take a look at the rdoc for the library.
rutema is a bit complex, so you might want to take a look at the documentation first.

To get the gem just do: gem install rutema

You can also get the gem directly from Rubyforge
