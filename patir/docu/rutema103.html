rutema 1.0.3 is out!

rutema 1.0.3 has been released!

After a couple of silent releases that fixed minor bugs we are proud to present rutema 1.0.3
1.0.3 is part of the evolutionary improvement of how rutema collects data from the tests. This release changes the way context information is stored in the DB (no schema upgrade necessary - it is all handled in code) to allow for easier access to that information.

Context information for a Run is now stored as a hash of values and rutemax adds three pieces of info:
The start time, the end time and the name of the configuration file.

v1.0.3 is accompanied by rutemaweb v0.9.2, which adds the aforementioned information to the Run overview pages and additionally adds failure rate percentages for each scenario.

To get the gems just do: gem install rutema
gem install rutemaweb

You can also get the gem directly from Rubyforge
