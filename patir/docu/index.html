patir

patir | rutema | rubot
Project Automation Tools in Ruby

patir aims to provide you with tools that make project automation (in Ruby) easier.
The patir project forms our basis for building project automation tools, an umbrella project housing the common code library and various subprojects.
Don't expect this library to grow very much, as we aim to keep it simple, stable and above all usable by all the tools under patir.

You can browse the patir subversion repository.

Like all self-conscious developers the patir team does not claim infalibility. Neither are we to proud to have our flaws pointed out, so if and when you find a bug, please post a report in our Bugtracker
