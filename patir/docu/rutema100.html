rutema 1.0.0 is out!

rutema 1.0.0 has been released!

It's finally here, rutema has reached v1.0.0!
Surprising little changes between 0.9.3 and 1.0.0 .
There is a DB schema change, so the 1.0.0 gem includes rutema_upgrader to facilitate upgrading existing databases. The migration is very simple.
The dependency to active record has been updated for verion 2.1.1 to stay current and the rutemah utility has been removed as it's functionality has been superseeded by rutemaweb.
Internally the parsers have been refactored to facilitate creating modules that provide ready made functionality - this will come in the form of a new gem rutema_elements which we will release in the coming days.

v1.0.0 is accompanied by rutemaweb v0.9.0, a major improvement of the web interface using Ramaze 2008.06

To get the gems just do: gem install rutema
gem install rutemaweb

You can also get the gem directly from Rubyforge
