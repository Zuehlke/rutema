rutema 0.8.0 is out!

rutema 0.8.0 has been released!

The refactoring knifes came out and plastic surgery was succesfully performed!

With this release rutemaweb splits from rutema. In order to get the web interface for rutema from now on you will need to do gem i rutemaweb.
This was done to allow us to evolve rutemaweb into a full fledged result repository. In the future rutemaweb will add a new reporter to rutema and the rutemaweb server will accept test run results from this reporter, thus serving as a central result repository.

Word of caution: it would be better to remove pervious rutema versions after installing v0.8.0 in order to remove the now incompatible rutemaweb versions.

To get the gems just do: gem install rutema gem install rutemaweb

You can also get the gem directly from Rubyforge
