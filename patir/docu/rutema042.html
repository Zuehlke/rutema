rutema 0.4.2 is out!

rutema 0.4.2 has been released!
Another bugfix release.

OK, 0.4.1 came out a bit early so a 0.4.2 is necessary to fix another couple of bugs in the minimal parser.
Also the distro test for the minimal parser didn't make it into the 0.4.1 gem so it is included in this one.
Also, note that there seems to be a problem with gem version 0.9.4 installations, probably because we upgraded to the newest Hoe for the 0.4.1 rutema release.
So you will need the latest and greatest gem (>0.9.5) for rutema to install via gems correctly.

To get the gem just do: gem install rutema

You can also get the gem directly from Rubyforge
