rutema_web 1.0.4 is out!

rutema_web 1.0.4 has been released!

It took a bit of time but it is finally here!

This release adds a few bugfixes, a bit of UI cleanup, the new timeline feature and configurable settings. It also brings rutema_web up-to-date with rutema 1.0.9 adding the ability to connect to any DB supported by activerecord.

The addition of the code that allows connection to DBs other than sqlite3 also brings a breaking change in the way rutema_web is called:
Instead of passing the db file, you now need a configuration file:

rutema_web [options] config.yaml
The configuration file is a yaml file in the following format: 
--- 
:db: 
  :adapter: sqlite3
  :database: rutema_big.db
:settings: 
  :page_size: 10
  :last_n_runs: 20
  :port: 7000
  :show_setup_teardown: true
The :db section corresponds to the activerecord database configuration (for those used to rails, it should be easy) and the above example is for sqlite3. The section for mySQL would look like: 
:db:
  :adapter: mysql
  :database: rutema
  :username: rutema
  :password: pass
  :host: meisterbrau.local

To get the gem just do: gem install rutema_web

You can also get the gem directly from Rubyforge
