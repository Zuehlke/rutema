rutema 1.2.0 is out!

rutema 1.2.0 has been released!

A much needed refactoring of the gem on the slow way to 1.9.x compatibility.
Updated gem dependencies to the latest available, added more unit tests and deprecated rutemax.
The new executable is now called simply 'rutema'.
This release is accompanied by a new version of the web fronted, rutema_web 1.0.5 which also updates the gem dependencies to the latest and greatest.

To get the gems just do:
gem install rutema

Note that the rutema gem does not depend on any particular adapter. You will have to have your favorite database set up and the appropriate gems installed for the active record reporter to work.

You can also get the gem directly from Rubyforge
