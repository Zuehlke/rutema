patir 0.5.3 is out!

Patir 0.5.3 has been released!

0.5.3 is a bugfix release that fixes an issue that arises when working directories are defined as relative paths and commands are used from code that changes directory after the command is defined.
Actually it's a bugfix for bugs introduced in the bugfix (0.5.1 and 0.5.2 are to be ignored at all costs)

Take a look at the rdoc for the library.
And since you're going to want to play with it...just do a

gem install patir

You can also get the gem directly from Rubyforge
