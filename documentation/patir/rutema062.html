rutema 0.6.2 is out!

rutema 0.6.2 has been released!

This is a bugfix release to fix rutemaweb (it lost it's commandline parsing along when moving to Ramaze and with all the changes it kinda slipped our notice) and add a way to pass information from the configuration to the parser. Stay tuned we will very shortly add documentation on the advanced configuration topics, such as how to define paths, pass context information and tool filenames so that they can be used in the parser and the reporter plugins.

To get the gem just do: gem install rutema

You can also get the gem directly from Rubyforge
