rutema_web 1.0.0 is out!

rutema_web 1.0.0 has been released!

It took a bit of time but it is finally here!

This release marks a major reworking of the code, switching from Ramaze to Sinatra.

The javascript used (as much as there is of it) has also been completely reworked and is now using jQuery.

Another subtle difference is that the library code does not use 'require "rubygems"' anymore. This makes testing a lot more consistent and in case you want to reuse the code keeps it from contaminating other libraries with gem dependencies.
The executable created by the gem installation will require rubygems and additionally lock down the required gems to the versions we have tested the code with so no additional work is needed when installing with 'gem'.

The name of the gem and the executable has changed in order to be more consistent with the rest of the rutema gems. It is now rutema_web

To get the gem just do: gem install rutema_web

rutema_web depends on gruff and it will pull the gruff gem as a dependency, but gruff does not define it's dependencies so you will probably have to install RMagick as well.
Don't worry, rutema_web will function even if gruff/RMagick are not properly installed - you just won't get the pretty colors.

You can also get the gem directly from Rubyforge
